# 心理支持助手 (Psychological Counselor)

一个 Claude Code 对话技能——让 AI 以同行者的姿态提供心理支持。不诊断、不开药、不代替治疗。只是陪一个人待一会儿。

> 我是一个纯粹的小白，这是我第一次用 vibe coding 做出来的东西。从想法到落地全在和 AI 吵——过程不专业、结构可能也不标准。如果你觉得哪里不对，直接说就好，但请轻喷 🙏

## 设计思路

大多数心理支持类 prompt 写成训练手册：阶段一二三四、决策表、操作流程。规则越细，程序感越强——来访者能感觉到对面那个人是在"运用技术"而不是"在陪你"。

这份 skill 换了一个写法：**散文体裁，同行者姿态。** 不教模型"执行什么步骤"，告诉它"站在哪里"。

核心框架保留了——四层深度、温和挑战、跨轮连接、危机转介——但规则嵌在散文里，不是列在标题下。读过之后更像一个人在你耳边说了几句话，而不是读完了一份操作手册。

## 能力边界

**可以做**：情绪支持、心理学知识科普、自助工具、陪伴梳理
**不碰**：诊断、开药、替代治疗、危机干预

遇到自杀想法/自伤行为、严重功能损害、疑似精神症状、物质滥用——会建议寻求线下专业帮助。

## 使用方法

1. 将整个目录放入 Claude Code 可访问的位置
2. 对话中表达情绪困扰或寻求心理支持时，skill 自动激活
3. 也可手动触发：`/psychological-counselor`
