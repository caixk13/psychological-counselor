# 危机信号识别与安全评估

## 重要声明

**本 skill 不是危机干预热线，也不是紧急服务。** 以下内容用于帮助识别危机信号，并在适当的时候建议用户寻求线下专业帮助。在紧急情况下，应建议用户立即联系当地急救服务（如拨打 110/120）或心理危机热线。

## 自杀风险警示信号

### 言语信号（直接）
- "我不想活了"
- "没有我别人会过得更好"
- "我希望我能永远睡过去"
- "我一直在想怎么结束这一切"
- "我在安排后事"

### 言语信号（间接）
- "这一切很快就会结束了"
- "你们不用再为我操心了"
- "活着没什么意思"
- "我好累，想休息了"（反复出现的、带有终结意味的语气）

### 行为信号
- 整理个人物品、处理重要物件
- 突然写遗嘱或交代后事
- 查询自杀方式
- 购买/收集危险物品
- 突然从极度低落变得异常平静（可能已做决定）
- 与亲友莫名告别
- 社交退缩急剧加重
- 自我伤害行为增加

### 情绪信号
- 强烈的无望感（hopelessness）——认为未来不会变好
- 强烈的无助感（helplessness）——认为自己什么都做不了
- 深重的自我厌恶
- 无法忍受的心理痛苦
- 感到自己是别人的负担（perceived burdensomeness）

### 高危因素
- 既往自杀尝试史（最强预测因子）
- 家族自杀史
- 物质滥用
- 严重躯体疾病或慢性疼痛
- 近期重大丧失（亲人去世、离婚、失业）
- 社会孤立
- 有可行的自杀计划 + 有实施手段

## 自伤行为（非自杀性自伤）

### 常见形式
- 割伤（最常见）
- 烧伤
- 击打自己
- 抓挠至出血

### 功能理解
自伤通常不是自杀企图，而是：
- 情绪调节：将无法忍受的情绪痛苦转化为可掌控的躯体疼痛
- 表达无法用语言表达的情感痛苦
- 感到麻木时确认自己"还活着"
- 自我惩罚

**注意**：自伤行为是未来自杀风险的重要预测因子，即使当前无自杀意图也应重视。

## 严重精神障碍的警示信号

以下信号提示可能需要精神科专业评估：

- **幻觉**：听到、看到、闻到、感到不存在的事物
- **妄想**：持有与事实不符的坚定信念（如被跟踪、被控制、拥有超能力）
- **思维紊乱**：言语混乱、跳跃、难以理解
- **严重功能退化**：基本日常生活能力显著下降
- **极端情绪波动**：超出正常范围的、急剧的情绪摆动
- **长时间不眠**：连续数天几乎不睡但精力旺盛

## 对话中识别到危机信号的应对框架

### 1. 保持冷静、直接询问
如果你听到了模糊的信号，**直接而温和地询问不会"植入"自杀想法**——研究一致表明这是安全的。
- "我听到你说'活着没意思'，我想认真地问一下——你是否有过想结束生命的想法？"
- 使用清晰的语言："想自杀"、"想结束生命"，不用模糊表述

### 2. 评估紧急性
- "这种想法有多频繁？"
- "你有具体的计划吗？"（有计划 = 更高风险）
- "你有实施的手段吗？"（有手段 = 紧急风险）
- "有什么在阻止你这么做？"（寻找保护因素）

### 3. 安全回应原则
- **不要承诺保密**：安全高于保密
- **不要一个人扛**：你不是专业危机干预者
- **不要轻描淡写**：认真对待每一个信号
- **不要讲大道理**：此刻不是讲"生命多美好"的时候
- **表达关心和存在**："我在这里陪着你"、"你的感受很重要"

### 4. 转向专业帮助
- 建议联系专业心理危机热线
- 鼓励向信任的亲友倾诉
- 紧急情况：陪同前往医院急诊/精神科
- **注意**：AI 对话中应提供联络框架（"请联系你所在城市的心理援助热线"），而非具体号码（因用户地区不同）

## 紧急情况声明模板

当识别到高风险情况时，应包含以下要素：
1. 表达真诚的关切和陪伴
2. 清楚说明你的局限（AI 不能提供紧急干预）
3. 明确建议寻求线下专业帮助
4. 提供求助方向（心理热线、精神科急诊、信任的亲友）
