# 认知行为疗法（CBT）核心技术

## 理论基础

认知行为疗法由 Aaron Beck 创立，核心假设是：**不是事件本身，而是个体对事件的认知（信念、想法）决定了情绪和行为反应。** 改变不适应性的认知模式，就能改变情绪和行为。

## 认知三角

```
         认知（想法）
          /        \
         /          \
    情绪（感受）---- 行为
```

三个顶点相互影响。切入点可以是认知（如质疑想法）、行为（如行为激活）、或情绪（如正念接纳）。

## 自动思维

自动思维是瞬间冒出的、不由自主的想法，当事人通常只感受到情绪而没注意到中间的思维过程。

**引导识别的话语**：
- "当那种感觉出现的时候，你脑子里闪过什么想法？"
- "在那个瞬间，你对自己/对这件事做了什么判断？"
- "那一刻你心里在跟自己说什么？"

## 常见认知扭曲

| 扭曲类型 | 描述 | 例子 |
|---------|------|------|
| **灾难化** | 预期最坏结果 | "这次没考好，我这辈子完了" |
| **非黑即白** | 极端二分思维 | "要么完美，要么失败" |
| **过度概括** | 从单一事件推出普遍规律 | "他又拒绝我了，从来没人喜欢我" |
| **心理过滤** | 只关注负面，忽略正面 | 收到9条好评却只纠结1条差评 |
| **情绪推理** | 把感觉当事实 | "我觉得自己很差劲，所以我一定很差劲" |
| **贴标签** | 用负面标签定义自己或他人 | "我是个失败者"、"他很自私" |
| **个人化** | 过度承担不属于自己的责任 | "他们不高兴一定是我做错了什么" |
| **"应该"暴政** | 僵化的规则和期望 | "我应该总是坚强的"、"别人应该理解我" |

## 苏格拉底式提问

不要直接告诉来访者"你的想法是扭曲的"，而是通过提问引导其自己发现：

1. **证据检验**："有什么证据支持这个想法？有什么证据不支持？"
2. **替代解释**："有没有其他的可能性/解释？"
3. **去灾难化**："如果最坏的情况真的发生了，结果会怎样？你能应对吗？"
4. **视角转换**："如果你的好朋友有同样的想法，你会怎么对他/她说？"
5. **影响评估**："持有这个想法对你有什么影响？放下它会怎样？"

## 行为激活

当来访者情绪低落、动力不足时，不等"有动力"再行动，而是**先行动，动力会随之而来**。

- **活动安排**：制定具体、可操作的小目标
- **愉悦感记录**：记录日常活动中实际体验到的愉悦感
- **渐进式目标**：从最容易的开始，积累成功体验

## 思维记录表（七栏表）

供来访者在情绪困扰时使用：

1. **情境**：什么时间/地点/发生了什么事？
2. **情绪**：你感受到了什么情绪？强度（0-100）？
3. **自动思维**：在情绪最强烈时脑子里闪过了什么想法？
4. **支持证据**：有什么事实支持这个想法？
5. **反对证据**：有什么事实不支持？有没有其他解释？
6. **平衡思维**：综合考虑证据后，更现实/平衡的想法是什么？
7. **结果**：现在情绪强度如何（0-100）？你打算做什么？

## 暴露与反应预防（ERP）

用于焦虑和强迫类问题。原则是：**在安全的环境中面对恐惧刺激，同时不做回避或仪式化行为。** 需要专业指导，在非专业对话中可介绍概念，但实践应由持证治疗师指导。

## 适用场景

- 焦虑障碍
- 抑郁情绪
- 社交恐惧
- 强迫思维
- 愤怒管理
- 低自尊
- 压力管理
