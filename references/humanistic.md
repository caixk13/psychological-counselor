# 人本主义心理学

## 理论基础

人本主义心理学由 Carl Rogers 和 Abraham Maslow 开创，被称为心理学的"第三势力"（区别于行为主义和精神分析）。核心理念：**每个人都有自我实现的内在倾向，在适当的条件下，人会自然地朝向成长、健康、自我实现的方向发展。**

## Rogers 的核心三条件

Rogers 认为，治疗师提供的这三个核心条件本身就是疗效因子，技术是次要的：

### 1. 共情（Empathy）

进入来访者的主观世界，**感受其所感受，同时保持"好像"的意识**——"好像"是来访者，但又不失去"我是我"的边界。

**共情的层次**：
- **初级共情**：准确反映来访者表达的感受和内容
  - "听起来你在工作中感到非常挫败和无力。"
- **深层共情**：捕捉到来访者自己尚未清晰意识到的感受
  - "我注意到每次提到你父亲时，你的语气变得很轻很轻，好像在小心地踏过一片薄冰——有没有一种可能的感受是，你在害怕让他失望？"

**共情的陷阱**：
- 不要急于给出建议（"你应该...试试"）→ 这会打断共情连接
- 不要换话题（"我也有类似的经历..."）→ 焦点从来访者转到了你身上
- 不要过度认同（"对！就是这样！太不公平了！"）→ 失去了"好像"的边界

### 2. 无条件积极关注（Unconditional Positive Regard）

**接纳来访者的全貌——包括矛盾、脆弱、黑暗面——不加评判，不附带条件。**

- 不因来访者的某些行为或感受而收回尊重
- 相信来访者的感受都有其原因和合理性
- 区分"接纳这个人"与"赞同某个行为"：我接纳你这个人，无条件；我对某些行为可能有不同看法

### 3. 真诚一致（Congruence / Genuineness）

治疗师在自己内部体验、意识、表达之间保持一致。**在适当的时候，治疗师也是真实的自己，而非躲在专业面具后面。**

- 不假装知道答案
- 如果被触动了，可以适当表达自己的感受
- 真诚不等于"什么都说"——表达要考虑是否服务于来访者的利益

## Maslow 需求层次

```
        自我实现
       /        \
    尊重需求（自尊、他人尊重）
   /            \
归属与爱的需求（关系、团体）
/                \
安全需求（稳定、安全、免于恐惧）
/                  \
生理需求（食物、水、睡眠）
```

提示：基本需求得不到满足时，人会卡在对应层次。理解来访者处于哪个层次，有助于判断干预重点。

## 此时此地（Here and Now）

关注当下正在发生的体验，而非沉溺于过去或过于忧虑未来：

- "此刻你说起这件事时，你的身体有什么感觉？"
- "现在，就在这个对话中，你感受到什么？"

## 自我实现倾向

Rogers 相信每个人内心都有朝向成长的积极力量。咨询师的角色不是"修理"来访者，而是**移除成长的障碍**——就像植物不需要被教怎么生长，只需要阳光、水和土壤。

**实践启示**：当来访者感到被困住时，可以问——"是什么在阻碍你成为你想成为的人？"而非"你应该怎么做？"

## 适用场景

- 自我认同困惑
- 关系问题
- 存在性焦虑（生命意义、方向感）
- 低自尊
- 成长性议题
- 任何需要深度信任与连接的心理工作
